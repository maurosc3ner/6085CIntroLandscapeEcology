1.1.1 Tiles

Lets start with a simple pie chart of the meningitis cases for a hospital:

library(leaflet)   # load the library
mymap<-leaflet() %>%
  addTiles # images you are going to see
mymap   #check the map

Now it is time to explore different tiles:

CartoDB:

mymap<-leaflet() %>%
  addProviderTiles(provider ="CartoDB.Positron")
mymap   #check the map

ESRI World Imagery:

mymap<-leaflet() %>%
  addProviderTiles(provider ="Esri.WorldImagery")
mymap   #check the map

Night lights:

mymap<-leaflet() %>%
  addProviderTiles(provider ="NASAGIBS.ViirsEarthAtNight2012")
mymap   #check the map
